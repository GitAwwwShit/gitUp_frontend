Owl Carousel

It is very easy to use Owl Carousel. No need to make any changes to CSS or JS. Just add your content inside the div tag with the class owl-content. Thats it. You are done. The content can be anything, image, text, etc., You can make changes to Carousel via data attributes data-items, data-auto-play, data-pagination, data-single-item.
